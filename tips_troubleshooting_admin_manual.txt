GETAROUNDMOBILITY ADMIN GUIDE
Tips and Troubleshooting Module

Last updated: 2026-05-19

==================================================
1) OVERVIEW
==================================================
The Tips and Troubleshooting admin page lets you manage:
- Section content (heading, description, slideshow images)
- Article content (title, description, optional image)
- Featured visibility (which articles appear in featured mode)
- Featured display order (drag-and-drop ordering)

Location:
- Admin URL: /admin/tips-troubleshooting

Role access:
- superadmin/admin: full edit access
- staff: view-only (cannot save changes)

==================================================
2) SECTION CONTENT (TOP LEFT CARD)
==================================================
Fields:
- Heading
- Description
- Image Alt Text
- Upload New Images (up to 5 shown in slideshow)

How it works:
1. Edit heading/description/alt text.
2. Optional: click Choose Images and select new files.
3. Click Save Section.

Important notes:
- New image upload replaces current slideshow images.
- If no image exists, default fallback image is used.
- Large uploads can fail if total size is too high.

==================================================
3) ADD ARTICLE (TOP RIGHT CARD)
==================================================
Fields:
- Title (required)
- Article Image (optional)
- Description (required)

How to add:
1. Enter title and description.
2. Optional: upload an article image.
3. Click Add Article.

Description formatting supported:
- Header token: *header size*
- Bold token: *bold*text*end of bold*
- Bullet token: *bullet*
- Numbered lines: 1. 2. 3.
- Markdown links: [label](https://example.com)
- Custom links: *link*label|https://example.com*end of link*

==================================================
4) EXISTING ARTICLES (PAGINATED LIST)
==================================================
Display behavior:
- Shows 3 articles per page in admin.
- Use page buttons at the bottom to move pages.

Per-article actions:
- Edit title/description/image and click Save Changes.
- Delete article with Delete button.
- Toggle Featured on/off using the switch.

Featured limit:
- Maximum featured articles allowed: 6
- If you try to feature a 7th article, admin shows an error.

==================================================
5) FEATURED DISPLAY ORDER
==================================================
Purpose:
- Controls the order of featured articles shown publicly.

How to use:
1. Feature desired articles first (up to 6).
2. In Featured Display Order panel, drag rows to reorder.
3. Click Save Display Order.

Tip:
- Order here determines top-to-bottom display in featured mode.

==================================================
6) PUBLIC PAGE BEHAVIOR (REFERENCE)
==================================================
Default public tips page:
- Shows featured articles only (up to 6)
- Uses your Featured Display Order

View all mode:
- Shows all articles (featured + non-featured)
- 15 articles per page with pagination

==================================================
7) IMAGE AND FILE HANDLING NOTES
==================================================
Section images:
- Multiple uploads allowed
- Keep image sizes reasonable for faster load

Article image:
- Optional per article
- Recommended landscape format for better display

==================================================
8) COMMON ISSUES AND FIXES
==================================================
Issue: "Staff account: view only"
Fix: Log in as admin or superadmin.

Issue: Cannot feature another article
Fix: You already have 6 featured; unfeature one first.

Issue: Save does not appear to change public order
Fix: In Featured Display Order, drag items then click Save Display Order.

Issue: Formatting not rendering as expected
Fix: Verify token syntax exactly (especially link/bold closing tokens).

Issue: Upload fails
Fix: Reduce file size/quantity and retry.

==================================================
9) QUICK WORKFLOW (RECOMMENDED)
==================================================
1. Add or update articles.
2. Toggle featured ON for the ones to show publicly.
3. Set featured sequence in Featured Display Order.
4. Save Display Order.
5. Verify on /tips-troubleshooting public page.

End of guide.
